# Python Calculator

Простой калькулятор с графическим интерфейсом (GUI), созданный на Python с использованием библиотеки Tkinter.

## Функционал
* Базовые арифметические операции (сложение, вычитание, умножение, деление).
* Очистка поля ввода (кнопка C).
* Удобный графический интерфейс.

## Как запустить проект локально
1. Убедитесь, что у вас установлен Python.
2. Скачайте файлы репозитория.
3. Запустите файл `calculator.pyw` двойным кликом или через терминал:
   ```bash
   python calculator.pyw
